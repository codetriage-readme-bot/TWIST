This branch was Created by Jonathan Pereira on 10th November 2015 for the TWIST platform. Open to the public domain.
Learn more about the sensor @ https://learn.adafruit.com/adafruit-si1145-breakout-board-uv-ir-visible-sensor/overview



This folder contains the Adafruit SI1145 Library, code and Schematics, Breadboard view  for the inclusion of the SI1145 sensor in the TWIST platform. The PCB view for this sensor is not correct.


Connections:
Connect Vin to the power supply, 5V is fine. For 3V microntrollers, connect to 3.3V
Connect GND to common power/data ground
Connect the SCL pin to the I2C clock SCL pin on your Edison. 
Connect the SDA pin to the I2C data SDA pin on your Edison. 



Read the ReadMe files in the library and code files too.
