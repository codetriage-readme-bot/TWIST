  In order to use this code for TWIST follow the follwing steps:
 * Step 1: Copy the DHT11 library in the arduino/libaries folder.
 * Step 2: Copy all the pin declarations, variable declarations,library initalizations to the main TWIST code before the setup block.
 * Step 3: Copy all the statement from the setup block into the setup block of the main TWIST code.
 * Step 4: Copy all the statement from the loop block into the loop block of the main TWIST code.
 * Step 5: Copy the ' void dht11(){.............} ' function and paste it outside the the loop block of the main TWIST code.
